Key Architecture Details:
- Region: us-east-1
- VPC with single availability zone
- ALB in public subnet (internet-facing)
- EC2 instances in private subnet (no auto-scaling)
- RDS MySQL in private subnet (SINGLE AZ - no Multi-AZ)
- S3 bucket with public access enabled
- NO encryption at rest configured
- NO CloudWatch monitoring
- NO backup strategy visible

This is intentionally a poor architecture for testing purposes!
